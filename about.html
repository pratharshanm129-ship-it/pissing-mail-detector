About PhishGuard

What is PhishGuard?

PhishGuard is a beginner-friendly web application built to help users identify potentially dangerous phishing emails. It was created as an educational project to demonstrate how a simple rule-based system can flag suspicious content without needing any complex machine learning model.

How Detection Works

Our detection engine scans the submitted email text for a curated list of common phishing "red flag" keywords and phrases, such as:

verify password urgent click here login bank gift winner account suspended free

The more matching keywords found, the higher the confidence that the email is a phishing attempt, and the higher the assigned risk level (Low, Medium, or High).

Educational Purpose

This project is intentionally kept simple so that students learning full-stack web development can understand every part of it -- from the Flask backend and SQLite database, to the Bootstrap-powered frontend. It is not intended to replace professional, enterprise-grade email security software.

Technology Stack
  • Backend: Python (Flask)
  • Database: SQLite
  • Frontend: HTML, CSS, JavaScript, Bootstrap 5
  • Authentication: Session-based login with hashed passwords
